# Xi-kki.github.io

> AI Agent & Web3 Builder Portfolio — [xi-kki.github.io](https://xi-kki.github.io)

Built with pure HTML, CSS, and JavaScript. Zero build steps, deployed via GitHub Pages.

## Sections

- **Hero** — Particle canvas, typewriter, social proof
- **Building With** — Tech stack strip
- **Projects** — VERIDD, The Xikki Brief, @Paiye_Bot, Zareb, Floe, Kraiiv
- **About** — Bio, stats, links
- **Writing** — The Xikki Brief newsletter archive
- **Journey** — Timeline of milestones
- **Contact** — Social links + email

## Features

- Dark / Light theme toggle (persisted to localStorage)
- Custom cursor with magnetic hover
- Particle network canvas background
- Typewriter effect in hero
- Tilt cards on project + stat cards
- Scroll-triggered reveal animations
- Animated stat counters
- Scroll progress bar
- Fully responsive

## Deploy

```bash
# Push to main branch — GitHub Pages auto-deploys
git add .
git commit -m "🚀 Initial portfolio"
git push origin main
```
